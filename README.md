# 💪 FitForge - Premium Gym & Member Tracking Portal

FitForge is a premium, fully responsive React web platform designed to present a modern fitness experience and provide members with high-performance tracking utilities. Built with **React 18**, **Vite**, and styled using custom **Vanilla CSS**, it features a sleek, dark-mode styling with energetic orange accents.

---

## 🌟 Key Features

### 🌐 Public Visitor Website
* **⚡ Home**: Cinematic landing experience with CTA sections, core benefits, and highlights.
* **🏋️ Fitness Programs**: Explore diverse training programs (e.g., strength training, HIIT, yoga, cardio) structured with visual previews.
* **🔥 Elite Trainers**: Meet certified fitness coaches, view their specializations, and schedule training.
* **📅 Class Schedule**: Dynamic day-by-day interactive gym schedule matching programs to time blocks.
* **💎 Membership Plans**: Clear tier-based pricing structure (Starter, Pro, Elite) with premium benefit listings.
* **📞 Contact Us**: Interactive contact detail hubs, business hours, and inquiry forms.
* **🚪 Join Now**: Direct signup entry gate for new members.

### 📊 Private Member Dashboard
* **🏠 Member Dashboard**: Personalized greetings, quick overview of daily goals, hydration, active time, and upcoming workouts.
* **💪 Workout Tracker**: Interactive workout planner to log sessions, track exercises, and monitor completion progress.
* **🥗 Nutrition Planner**: Meal planning workspace with macronutrient trackers (carbs, protein, fats, calories) and meal logs.
* **📈 Performance Analytics**: Visual analytics charting calories burned, active minutes, and weight goals over time.
* **⚙️ Account Settings**: Member profile management, notifications toggle, fitness goals customizers, and security settings.

## 🛠️ Technology Stack

* **Frontend Framework**: React 18
* **Build System**: Vite (Fast HMR & Optimized Bundles)
* **Routing**: React Router DOM (Dynamic routing with split layouts)
* **Styling**: Vanilla CSS (Custom Design Token System in [index.css](file:///d:/CodeSrishti/FitForge/src/index.css))
* **Icons & Fonts**: Google Fonts (Montserrat & Inter) & Material Symbols Outlined

---

## 🎨 Design System & Styling

FitForge utilizes a sleek **Dark & Energetic Accent** styling system configured using CSS Variables in [index.css](file:///d:/CodeSrishti/FitForge/src/index.css):
* **Primary Accent**: Energetic Orange (`--accent-orange: #FF6B00`)
* **Background Color**: Dark Charcoal (`--background: #0F0F0F`)
* **Surface Containers**: Dark Slates (`--surface: #121414`, `--surface-container: #1e2020`)
* **Typography**: Montserrat (Headlines) & Inter (Body Copy)
* **Animations & Glows**: Subtle, high-end effects like glassmorphic cards (`.glass-card`), glowing shadows (`.accent-glow`), pulsing progress bars (`.progress-pulse`), and smooth translate-lifts.

## 🚀 Getting Started

Follow these steps to run the FitForge project locally:

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed (version 16.x or higher is recommended).

### Installation

1. Navigate to the project directory:
   ```bash
   cd FitForge
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

### Running Locally

To launch the local development server:
```bash
npm run dev
```
Open your browser and navigate to the local address displayed in your terminal (usually `http://localhost:5173`).
